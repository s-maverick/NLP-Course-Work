# Advanced NLP Course Work

## University of Massachusetts, Amherst [COMPSCI 685]

This repository hosts a collection of Natural Language Processing (NLP) projects and lab assignments, focusing on the computational analysis and modeling of human language. The coursework covers the progression from classical text preprocessing and linguistic rule-based systems to modern deep learning architectures, including sentiment analysis, named entity recognition (NER), and large language model (LLM) fine-tuning.

### Tech Stack & Tools

* **Languages:** Python (Core development), SQL (Data retrieval).
* **NLP Frameworks:** NLTK, Spacy, Hugging Face Transformers.
* **Deep Learning:** PyTorch, TensorFlow, Keras.
* **Vectorization & Embeddings:** Word2Vec, GloVe, BERT, Sentence-Transformers.
* **Utilities:** Scikit-learn (Evaluation), Gensim, Regular Expressions (RegEx), Jupyter Notebooks and Google Colab.
